## 操作系统基本概念


### 概述
    - 计算机硬件结构：cpu 内存 IO控制单元；
    - 早期的时候基本上每个io设备都有 I/O控制单元；
    - 2004年开始，摩尔定律不再明显，intel开始牙膏；
      - 为继续提高处理器性能，出现了SMP 对称多处理器 symmetrical multi-processor
      - 处理器中的缓存器件成本高，出现了MP 多核心处理器 multi-core processor 多个核心共享缓存 很像大学里用的28377；

### 提高cpu利用率
    - 在单片机时代，我们单核单线程，逻辑简单相对低效；
    - 其实在dpdk中，我们又有点回到了原来的这种状态，因为任务切换有开销；
    - 发展：最早为 多道程序 分时系统 多任务系统；
    - 操作系统的功能 提供抽象接口 管理硬件资源
    - 多任务系统其实核心是想实现：每个进程至少在逻辑上独立占有各种资源
### 线程
    - thread lightweightprocess lwp
    - 线程组成 线程id 内存空间
